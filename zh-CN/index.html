基于 Promise 的 API

Agora Web SDK NG 现在使用基于 Promise-based API 来帮助您编写更好的异步操作代码。现在可以试试在 Agora Web SDK 上使用 async/await！

支持 Typescript

Agora Web SDK NG 现在使用 Typescript 开发。 我们提供了一个完整的声明文件，以便您可以体验自动补全、类型检查和更多其他 Typescript 功能。

更易用

我们对部分 API 做了调整，以确保用户能有更好的开发体验。

快速上手

配合现代 Javascript 和我们的 SDK，你可以只使用几行代码就快速上手。查看右侧的示例代码！

功能对齐

Agora Web SDK NG 相对于 Agora 官方的 Web SDK（Agora Web SDK v3）实现了功能对齐 。 支持图像增强、混音、直播、跨频道以及其他所有官方 SDK 所支持的功能。

完善的 Demo 和示例代码

对于每个常用功能，我们都提供了对应的 Demo，比如基本音视频通话、调整视频属性、采集设备控制等等。 您可以在线上访问我们 Demo 页面，同时，每个 Demo 的源码我们都放在了 Github 上。
