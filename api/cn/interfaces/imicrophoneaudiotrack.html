Interface IMicrophoneAudioTrack

本地麦克风音频轨道，继承于 LocalAudioTrack，在此之上提供了更换采集设备的功能。

可以通过 AgoraRTC.createMicrophoneAudioTrack 来获取。

Properties

isMuted

isMuted: boolean

Inherited from ILocalTrack.isMuted

本地轨道是否 Mute：

  • true: 该轨道 Mute。
  • false: 该轨道 Unmute。

Mute 后，本地轨道依然会发送音频/视频数据：

  • Mute 视频轨道后，发送黑帧。
  • Mute 音频轨道后，发送静音帧。

isPlaying

isPlaying: boolean

Inherited from ITrack.isPlaying

媒体轨道是否正在页面上播放。

  • true: 媒体轨道正在页面上播放。
  • false: 媒体轨道没有在页面上播放。

trackMediaType

trackMediaType: "audio" | "video"

Inherited from ITrack.trackMediaType

媒体轨道的类型：

  • "audio": 音频轨道。
  • "video": 视频轨道。

getTrackLabel

  • getTrackLabel(): string

  • Inherited from ILocalTrack.getTrackLabel

    获取本地轨道的来源描述。

    Returns string

    可能返回以下值：

    • 如果是通过 createMicrophoneAudioTrack 或 createCameraVideoTrack 创建的轨道，返回 MediaDeviceInfo.label 字段。
    • 如果是通过 createScreenVideoTrack 创建的轨道，返回屏幕共享的 sourceId。
    • 如果是 createCustomAudioTrack 或 createCustomVideoTrack 创建的轨道，返回 MediaStreamTrack.label 字段。

getUserId

  • Inherited from ILocalTrack.getUserId

    Returns null | UID

    该本地轨道对应的 uid。

  • once(event: string, listener: Function): void

  • Inherited from EventEmitter.once
