Interface ILocalTrack

ILocalTrack 为本地轨道的基础类，为音频轨道 LocalAudioTrack 和视频轨道 LocalVideoTrack 提供一些公共的方法。

Events

track-ended

  • track-ended(): void

  • 音频/视频轨道被终止，终止的原因可能是以下情况：

    • 摄像头被拔出
    • 麦克风被拔出
    • 用户主动停止了屏幕共享
    • 该轨道被主动关闭
    • 媒体设备故障

    Returns void

Properties

isMuted

isMuted: boolean

TBD 标记该轨道是否被 mute

  • 对于视频流：mute 指黑屏
  • 对于音频流：mute 指静音

标记该轨道是否禁用。

returns
  • true: 该轨道禁用。
  • false: 该轨道未禁用。

isPlaying: boolean

trackMediaType

trackMediaType: "audio" | "video"

Inherited from ITrack.trackMediaType

表示媒体流的类型，音频流还是视频流

close

  • close(): void

  • 关闭本地流，并释放相关采集设备。

    一旦本地流被关闭，就无法再次使用。如需再次使用本地流，需要重新创建。

    Returns void

once

  • once(event: string, listener: Function): void

  • Inherited from EventEmitter.once

    监听一个指定的事件，当事件触发时会调用传入的回调函数。

    当监听后事件第一次触发时，该监听和回调函数就会被立刻移除，也就是只监听一次指定事件。

    Parameters

    • event: string

      指定事件的名称。

    • listener: Function

      传入的回调函数。

    Returns void
