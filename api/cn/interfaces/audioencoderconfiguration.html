Interface AudioEncoderConfiguration

定义音频编码配置的对象。

用于创建音频流时指定自定义的编码配置。

你可以在 AgoraRTC.createCustomAudioTrack、 AgoraRTC.createMicrophoneAudioTrack 或 AgoraRTC.createBufferSourceAudioTrack 方法中传入该配置对象来自定义本地音频的编码配置。
