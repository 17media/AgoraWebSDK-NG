Interface ICameraVideoTrack

本地摄像头视频流，继承于 LocalVideoTrack，在此之上提供了更换采集设备/调整编码参数的功能。

可以通过 AgoraRTC.createCameraVideoTrack 来获取。

once

  • once(event: string, listener: Function): void

  • Inherited from EventEmitter.once

    监听一个指定的事件，当事件触发时会调用传入的回调函数。

    当监听后事件第一次触发时，该监听和回调函数就会被立刻移除，也就是只监听一次指定事件。

    Returns void

play

  • play(element: string | HTMLElement, config?: VideoPlayerConfig): void

  • Inherited from ILocalVideoTrack.play

    Overrides ITrack.play

    在页面上播放视频流

    Parameters

    • element: string | HTMLElement

      指定一个 DOM 元素，SDK 将在这个元素下创建 <video> 元素播放视频流，支持 2 种类型：

      • string: 指定该 DOM 元素的 id 值
      • HTMLElement： 直接传入一个 DOM 元素对象
    • Optional config: VideoPlayerConfig

      设置播放参数（镜像/显示模式），对于本地视频流，镜像默认开启。

    Returns void

setBeautyEffect

  • setBeautyEffect(enabled: boolean, options?: BeautyEffectOptions): Promise<void>

  • Inherited from ILocalVideoTrack.setBeautyEffect

    开启或关闭美颜。

    注意事项：

    • 该方法支持以下浏览器：
      • Safari 12 及以上版本。
      • Chrome 65 及以上版本。
      • Firefox 70.0.1 及以上版本。
    • 该功能不支持移动端设备。
    • 如果开启了双流模式，美颜选项仅对大流生效。

    Parameters

    • enabled: boolean

      是否开启本地美颜功能

    • Optional options: BeautyEffectOptions

      美颜效果选项

    Returns Promise<void>
