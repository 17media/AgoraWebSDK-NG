AgoraWebSDK NG API Docs

Agora Web SDK NG API Docs

The Agora Web SDK NG (WebRTC) is a JavaScript library loaded by an HTML web page. The Agora Web SDK NG library uses APIs in the web browser to establish connections and control the communication and live broadcast services.

Ensure that you use the HTTPS protocol or localhost, otherwise the error WEB_SECURITY_RESTRICT occurs.

Suggestions
  • We recommend reading the Basic Features Documentation before reading the API details.
  • You can use the search button at the top of the page to quickly search for the API you want to see

Basic Modules

AgoraRTC is the entry point for all the methods that can be called in Agora Web SDK NG.

AgoraRTC provides the following methods:

Core Methods
Method Description
createClient Create a local client object

Local Track Methods
Method Description
createMicrophoneAudioTrack Creates an audio track from the audio captured by a microphone
createCameraVideoTrack Creates an video track from the video captured by a camera
createScreenVideoTrack Creates a video track for screen sharing
createBufferSourceAudioTrack Creates an audio track from an audio file
createCustomAudioTrack Creates a custom audio track
createCustomVideoTrack Creates a custom video track

Media Devices Methods
Method Description
getDevices Enumerates the media input and output devices available
getCameras Enumerates the video input devices available, such as cameras
getMicrophones Enumerates the audio input devices available, such as microphones
getElectronScreenSources Gets the sources for screen-sharing on Electron

Logger Methods
Method Description
enableLogUpload Enables log upload
disableLogUpload Disables log upload
setLogLevel Sets the output log level of the SDK

Other Methods
Method Description
checkSystemRequirements Checks the Web browser compatibility
getSupportedCodec Gets the supported codecs of the web browser
createChannelMediaRelayConfiguration create a ChannelMediaRelayConfiguration object

How to import

If you use <script> to import Agora Web SDK NG, you can call window.AgoraRTC to get the export methods.

If you use npm to import Agora Web SDK NG, just import it:

import AgoraRTC from "agora-web-sdk-ng"
console.log(AgoraRTC.createClient);

Typed Modules (For Typescript)

 
import AgoraRTC, { IAgoraRTCClient } from "agora-web-sdk-ng"

const client: IAgoraRTCClient = AgoraRTC.createClient()
