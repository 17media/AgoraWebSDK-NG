Interface ICameraVideoTrack

ICameraVideoTrack is the interface of local video captured by a camera, which inherits the LocalVideoTrack interface and adds some functions such as switching devices and adjust video encoder configurations.

You can create a local camera video track by calling the AgoraRTC.createCameraVideoTrack method.

Properties

isMuted

isMuted: boolean

Inherited from ILocalTrack.isMuted

Marks whether the track is muted.

After a local track is muted, it still sends audio or video data:

  • A muted local audio track sends silenced audio frames.
  • A muted local video track sends black video frames.
returns

The mute state of the track:

  • true: The track is muted.
  • false: The track is unmuted.

setBeautyEffect

  • setBeautyEffect(enabled: boolean, options?: BeautyEffectOptions): Promise<void>

  • Inherited from ILocalVideoTrack.setBeautyEffect

    Enables/Disables image enhancement and sets the options.

    Notes:

    • This method supports the following browsers:
      • Safari 12 or later.
      • Chrome 65 or later.
      • Firefox 70.0.1 or later.
    • This function is supported on mobile devices.
    • If the dual-stream mode is enabled, the image enhancement options apply only to the high-video stream.

    Parameters

    • enabled: boolean

      Sets whether to enable image enhancement:

      • true: Enables image enhancement.
      • false: Disables image enhancement.
    • Optional options: BeautyEffectOptions

      Sets image enhancement options.

    Returns Promise<void>
