Interface IMicrophoneAudioTrack

IMicrophoneAudioTrack is the interface of local audio captured by a microphone, which inherits the LocalAudioTrack interface and adds some functions such as switching devices.

You can create a local microphone audio track by calling the AgoraRTC.createMicrophoneAudioTrack method.

Properties

isMuted

isMuted: boolean

Inherited from ILocalTrack.isMuted

Marks whether the track is muted.

After a local track is muted, it still sends audio or video data:

  • A muted local audio track sends silenced audio frames.
  • A muted local video track sends black video frames.
returns

The mute state of the track:

  • true: The track is muted.
  • false: The track is unmuted.
