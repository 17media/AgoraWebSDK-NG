Promise-based API

Agora Web SDK NG now uses the standard Promise-based API to help you write better async operations. Try using Agora Web SDK with async/await !

Typescript Support

Agora Web SDK NG is now based on Typescript. We provide a complete declaration file so you can enjoy autocompletion, type checking and more typescript features.

Easier to use

We adjusted some APIs to ensure a better development experience for users.

Get Started Quickly

With modern Javascript and our SDK, you can get started quickly with a few lines of codes. Check out the sample codes in the right!

Full-Featured

Agora Web SDK NG is a full-featured SDK with the Agora official version web SDK (Agora Web SDK v3). It supports Image Enhancement, Audio Mixing, Live Streaming, Co-host across Channels and all features supported by the official web SDK

Demos and Sample Codes

For each common used feature, we provide a corresponding demo page, such as basic video call, adjust video profile, recording device control and so on. You can check out our online demos, we put the source codes of each demo on the Github.
